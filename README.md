# 📱 User List App

Bu proje, **Turkcell Geleceği Yazanlar Android Bootcamp** kapsamında geliştirilmiştir. Modern Android geliştirme standartlarına uygun olarak tasarlanan uygulama, bir REST API üzerinden kullanıcı verilerini asenkron olarak çekmekte ve kullanıcı dostu bir arayüz ile sunmaktadır.

---

## 🚀 Öne Çıkan Özellikler

* **MVVM Mimarisi:** Temiz kod prensipleriyle ayrıştırılmış Model-View-ViewModel katmanları.
* **Jetpack Compose:** Tamamen deklaratif ve modern UI bileşenleri.
* **Retrofit & Coroutines:** `jsonplaceholder.typicode.com` üzerinden performanslı ve güvenli veri çekimi.
* **Anlık Arama (Search):** Kullanıcı listesinde isim bazlı filtreleme yapabilen dinamik arama çubuğu.
* **Gelişmiş State Yönetimi:** * Veri çekilirken **Loading** (Yükleme) durumu.
    * Hata anında (İnternet yoksa vb.) bilgilendirici **Error** mesajı ve "Tekrar Dene" butonu.
* **UX Detayları:** `statusBarsPadding` kullanımı ile kamera deliği (notch) ve sistem çubuklarına duyarlı tasarım.

---

## 🛠 Kullanılan Teknolojiler

- **Dil:** Kotlin
- **Asenkron Yapı:** Coroutines & Flow
- **UI:** Jetpack Compose & Material 3
- **Network:** Retrofit & OkHttp
- **Mimari:** MVVM & State Management
